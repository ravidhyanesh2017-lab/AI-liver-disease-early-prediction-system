# AI Liver Disease Early Prediction System

This project is a web-based clinical decision support system for early liver health assessment.  
Users can enter biochemical parameters such as bilirubin, SGPT, SGOT, ALP, total protein, and albumin to receive an early liver health assessment.

## Technologies Used
Java | Spring Boot | HTML | CSS

## Features
- Patient biochemical data input form
- Automated liver health assessment
- Web-based user interface
